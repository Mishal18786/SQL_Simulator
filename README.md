# SQL_Simulator
The SQL Query Simulator is a software application which is designed to provide users an interactive application to execute SQL Query with MYSQL database
Applications: 
  Python 3.10 or later 
  MySQL 8.0 
  MySQL connector ODBC 8.3.0
Python Packages:
  Pyside6
  Qt-material
  Pandas
  Sys
  PYODBC
  MySQL-connector
  PYODBC
